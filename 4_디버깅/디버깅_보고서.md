## 🐛 발견한 오류 #1

### 1) 어떤 문제가 발생했나요?
프로그램 실행이 안되었다.(실행자체가 안됨)

```
문제 상황을 간단히 설명하세요.
```

### 2) 오류가 발생한 코드
```c
// 문제가 있던 코드를 여기에 복사하세요
 struct Student {
        char name[50] 
        int attend;
        int totalClass;
        int excused;
        double rate;
```

### 3) 왜 오류가 생겼나요?
 char name[50] 부분에 세미콜론을 빼먹었다.

```
원인을 간단히 적어보세요.
```

### 4) 어떻게 고쳤나요?
```c
 struct Student {
        char name[50];
        int attend;
        int totalClass;
        int excused;
        double rate;
```

### 5) 고친 후 결과
- [O] ❌ 아직 문제가 있어요

## 🐛 발견한 오류 #2

### 1) 어떤 문제가 발생했나요?
```
변수가 존재하지 않는다는 문구가 출력되었다.
```

### 2) 오류가 발생한 코드
```c
 s.rate = (s.attend + s.excused) / total * 100;
```

### 3) 왜 오류가 생겼나요?
```
tatal이라는 변수를 선언한적이 없었다. s.totalClass를 사용했어야 한다.
```

### 4) 어떻게 고쳤나요?
```c
s.rate = (s.attend + s.excused) / s.totalClass * 100;
```

### 5) 고친 후 결과
- [o] ❌ 아직 문제가 있어요

## 💭 디버깅하면서 배운 점

이번 프로젝트에서 오류를 고치면서 새롭게 알게 된 점이나 느낀 점을 자유롭게 적어보세요.

```
(예시)
- 세미콜론(;)을 빼먹으면 컴파일 오류가 난다는 것을 배웠어요.
- scanf를 쓸 때 &를 꼭 써야 한다는 걸 알았어요.
- 에러 메시지를 잘 읽으면 문제를 더 쉽게 찾을 수 있었어요.
```
