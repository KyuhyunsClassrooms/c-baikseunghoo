# 1. 프로그램 설계서 (5점)

## 1. 프로젝트 주제
출석 일수에 따른 이수/미이수 여부 판독 프로그램

## 2. 핵심 기능 목록

1.  학생의 이름과 출석일수를 입력받음
2.  총 수업 일수를 기준으로 출석률을 게산한다.
3.  출석 인정 결석 일수를 물어본다
4.  출석률이 일정 기준 이상이면 이수, 기준 미만이면 미이수를 출력한다.

## 3. 데이터 구조 설계
(프로그램에 필요한 핵심 변수, 배열을 설계하세요. 필요하면 구조체(struct)를 사용할 수도 있습니다.)

**방법 2: 구조체 사용**
#include <stdio.h>

    // 학생 정보를 담기 위한 구조체 타입 Student 정의
    // 이름, 출석일수, 총 수업일수, 인정 결석일수, 출석률 저장
    struct Student {
        char name[50];     // 학생 이름
        int attend;        // 실제 출석 일수
        int totalClass;    // 총 수업 일수
        int excused;       // 출석 인정 결석 일수
        double rate;       // 계산된 출석률
    };
