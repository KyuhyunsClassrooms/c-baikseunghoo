# 1. 프로그램 설계서 (5점)

## 1. 프로젝트 주제
출석 일수에 따른 이수/미이수 여부 판독 프로그램

## 2. 핵심 기능 목록

1.  학생의 이름과 출석일수를 입력받음
2.  총 수업 일수를 기준으로 출석률을 게산한다.
3.  출석률이 일정 기준 이상이면 이수, 기준 미만이면 미이수를 출력한다. 

## 3. 데이터 구조 설계
(프로그램에 필요한 핵심 변수, 배열을 설계하세요. 필요하면 구조체(struct)를 사용할 수도 있습니다.)

**방법 2: 구조체 사용**
#include <stdio.h>

// 학생 정보를 저장하기 위한 구조체 정의
struct Student {
    char name[20];     // 학생 이름
    int attendance;    // 출석 일수
};

int main() {
    // 총 수업일수와 이수 기준 정의
    int total_classes = 365; // 총 수업일수 (예: 365일)
    int pass_criteria = 300;  // 이수 기준 (예: 300일 이상 출석 시 이수)

    // 학생들의 이름과 출석 일수를 구조체 배열로 저장
    struct Student students[5] = {
        {"홍길동", 209},
        {"김철수", 330},
        {"이영희", 337},
        {"박민수", 117},
        {"최수지", 345},
    };
